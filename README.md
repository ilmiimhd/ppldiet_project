# 🥗 DIETIFY (Healthy Diet Planner App)

A web-based application designed to help users manage a structured 30-day diet plan, including recommendations, tracking, and scheduling.

---

## 📌 Project Overview
This project was developed as a course assignment to design and build a diet planning application that helps users follow a healthy lifestyle through structured meal plans and tracking features.

## 🧠 System Development
The system was developed following SDLC approach:

- **Analysis**: Research on healthy diet & user needs  
- **Design**: UML diagrams (Use Case, Activity, ERD) + UI/UX prototype  
- **Implementation**: Frontend & backend development  
- **Testing**: Functional testing based on predefined scenarios  
- **Deployment & Maintenance**: System improvement based on evaluation
